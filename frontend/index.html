SolanaTxPlain

Real-time Solana activity explainer — enter a wallet, perform actions, see what just happened in plain English.

About

SolanaTxPlain is a real-time Solana activity explainer. Instead of pasting one transaction, you enter a wallet address. We listen to live Solana transactions for that wallet, group bursts (1–3 seconds) into a single action, and use AI to explain what just happened in plain English.

Solana produces many transactions in milliseconds. We translate that fast, noisy activity into one human-readable story per burst — with fees, outcome, and why multiple txs are normal on Solana (fast finality, low fees).

How to use

1
Choose network

Pick Mainnet (real SOL) or Devnet (test SOL). Devnet is free and perfect for trying the app.

2
Enter a wallet address

Paste a Solana public key. This is the wallet whose activity we'll explain.

3
Click Start live

We subscribe to Solana and listen for transactions that mention this wallet.

4
Perform actions → see explanations

Send SOL, swap tokens, mint an NFT, etc. Explanations appear in the Live Activity feed as txs finalize.

Use Solana Devnet (perfect for testing)

Devnet gives you free test SOL and real transactions so you can try Live Activity without spending real funds.

1
Switch to Devnet

Select Devnet in the network dropdown below. We use RPC: https://api.devnet.solana.com

2
Create a test wallet

Run solana-keygen new. You get a pubkey and a private key (keep it local).

3
Get free SOL

Run solana airdrop 2 (with Solana CLI configured for devnet).

4
Generate activity

Send SOL: solana transfer <OTHER_PUBKEY> 0.01. Your app listens and explains in real time.

You get real transactions, instant confirmations, and full control — with no mainnet risk.
